# 웹 배포 가이드 (무료)

## 필요한 계정 (모두 무료)
- GitHub: https://github.com
- Supabase: https://supabase.com
- Streamlit Cloud: https://share.streamlit.io

---

## STEP 1 — Supabase 데이터베이스 설정

1. https://supabase.com 접속 → **Start your project** → 회원가입/로그인
2. **New project** 클릭 → 프로젝트 이름 입력 (예: `재고모니터`) → DB 비밀번호 설정 → Create
3. 왼쪽 메뉴 **SQL Editor** 클릭 → 아래 SQL 실행:

```sql
CREATE TABLE watchlist (
  id     SERIAL PRIMARY KEY,
  code   VARCHAR(50) NOT NULL,
  expiry VARCHAR(8)  NOT NULL,
  UNIQUE(code, expiry)
);
```

4. 왼쪽 메뉴 **Settings** → **API** 클릭
   - **Project URL** 복사 → `SUPABASE_URL`
   - **anon / public** 키 복사 → `SUPABASE_KEY`

---

## STEP 2 — GitHub에 코드 올리기

1. https://github.com/new 에서 **private** 저장소 생성 (이름 예: `jaegomonitor`)
2. 이 폴더(`재고모니터`) 전체를 해당 저장소에 push:

```bash
cd 재고모니터
git init
git add .
git commit -m "init"
git remote add origin https://github.com/내ID/jaegomonitor.git
git push -u origin main
```

---

## STEP 3 — Streamlit Cloud 배포

1. https://share.streamlit.io 접속 → GitHub 계정으로 로그인
2. **New app** 클릭
   - Repository: `내ID/jaegomonitor`
   - Branch: `main`
   - Main file path: `app.py`
3. **Advanced settings** → **Secrets** 탭에 아래 입력:

```toml
SUPABASE_URL = "https://xxxx.supabase.co"
SUPABASE_KEY = "eyJ..."
```

4. **Deploy** 클릭 → 1~2분 후 URL 생성됨

---

## 완료!

생성된 URL (예: `https://내이름-jaegomonitor.streamlit.app`)을  
직원들에게 공유하면 누구나 인터넷으로 접속 가능합니다.

### 접속 방법
- 품목 등록/삭제: **"⚙️ 품목 등록/삭제 열기"** 체크박스 클릭
- 파일 분석: Excel 파일 업로드

### 데이터 관리
- 워치리스트: Supabase 대시보드 → Table Editor → watchlist 에서 직접 확인/수정 가능
